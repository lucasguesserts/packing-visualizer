# Packing Visualizer

A simple web-based tool to visualize packing of rectangular items.

## How to Run

```sh
npm install
npx start
```

## CLI tools

```sh
npm link
check-output OUTPUT_FILE
analyse-data RESULTS_OUTPUT_DIR CSV_FILE_PATH_TO_SAVE
```
